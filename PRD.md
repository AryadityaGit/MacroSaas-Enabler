## Purpose
Enable users to run Excel macros offline without Excel or macro permissions.

## Features
- Upload macro Excel file
- Upload target Excel file
- Simulate macro logic (XLOOKUP, autofill)
- Download updated file
- Display ghostly macro button based on uploaded `.xlsm`

## Constraints
- Must work offline
- No installation required
- No Excel dependency
- Must be beginner-friendly

## Future Scope
- Support more macro commands
- Add folder scanning
- Add local model for smart macros
