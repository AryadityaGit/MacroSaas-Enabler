# MacroSaas Enabler — VBA Ghost

## What is this?
A simple offline tool that runs Excel macros without needing Excel. You upload your macro file and target file, and it applies logic like XLOOKUP.

## How to use
1. Open `VBA-Ghost.html` in your browser (Chrome or Edge).
2. Upload your macro Excel file (`.xlsm` or `.xlsx`).
3. Upload your target Excel file.
4. Click the ghostly macro button.
5. A new file will be downloaded with the updated data.

## Folder Setup
Create a folder named `MacroSaas-Enabler` and add:
- `VBA-Ghost.html`
- `README.md`
- `PRD.md`
- `COST.md`

## No coding needed
Just copy and paste the files. You don’t need to install anything or write code.
